Bikes Sales Project

Introduction

This project is part of Alex the Analyst's online excel course for beginners, the data and procedures were made to implement the learning obtained.

Development
  • I was able to use the Excel tools to be able to remove some values that were not necessary for the dashboard.
  • For the elaboration of the graphs, I used the PivotTable tool to extract the fields that I considered pertinent to show to those in charge of the marketing department.
  • The last part involved placing the relevant charts on the dashboard, applying some formatting and layout adjustments, and creating slices to make the chart interactive with some important parameters, such as marital status, region, and professional occupation.
